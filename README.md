### Smart Financial EMI Estimator · Interactive Graphs & Projections

**Loan EMI Calculator** is an interactive browser tool designed to compute Equated Monthly Installments (EMI) for various loans. Featuring clear financial indicators and slide-to-update metrics.

*Instant Mathematical Projections · Fully Client-side · Responsive CSS Glassmorphism*

</div>

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🧮 **Accurate EMI Estimation** | Instantly calculates EMI using standard interest amortization algorithms. |
| 💫 **Real-time Interaction** | Sliders adjust Principal Amount, Rate of Interest, and Tenure instantly. |
| 📈 **Accrued Interest Breakdown** | Clearly separates the principal loan amount from total interest payable. |
| 📱 **Responsive Design** | Optimized interface that scales smoothly across mobile, tablet, and desktop views. |

---

## 🧱 Tech Stack

* **Structure**: Semantic HTML5 layout tags
* **Styling**: Vanilla CSS (Responsive Flexbox and grid classes)
* **Calculations**: Vanilla JavaScript (Standard EMI loan amortization mathematical models)

## ⚙️ Running Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ParthaG23/Loan_EMI_Calculator.git
   ```
2. **Open the project**:
   - Double click `index.html` to launch the EMI tool locally in any browser.
